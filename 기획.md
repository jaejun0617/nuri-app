### 📋 반려견 기록 앱 상세 기획 정리

#### 1. 사용자 맞춤형 인사말 (Supabase 연동)

- **기능:** 로그인 후 `"{닉네임}님, 반가워요!"` 문구 노출.
- **구현:** Supabase Auth 가입 시 `user_metadata`에 저장된 닉네임을 불러오거나, 별도의 `profiles` 테이블을 참조하여 실시간으로 반영합니다.
- **효과:** 사용자에게 개인화된 느낌을 주어 서비스 친밀도를 높입니다.

#### 2. 프로필 이미지 동기화

- **기능:** 반려견 등록 시 업로드한 사진이 메인 프로필과 우측 상단 미니 프로필에 동시 적용.
- **구현:** Supabase Storage에 이미지를 저장하고 해당 URL을 DB에 저장합니다. 전역 상태 관리(Redux, Recoil 등)를 통해 한 번의 수정으로 앱 전체의 프로필 아이콘이 업데이트되도록 설계합니다.

#### 3. 반려견 정보 등록 및 디데이(D-Day) 계산

- **등록 항목:** 이름, 생년월일, 몸무게, 좋아하는 것(#태그), 싫어하는 것(추가 추천), 취미.
- **날짜 계산:**
  - '함께한 시간' 자동 계산: `현재 날짜 - 등록한 기념일/생일`을 계산하여 **"누리와 함께한 지 500일째"** 같은 문구를 메인에 노출.
- **💡 아이디어 추가:**
  - **건강 데이터:** 마지막 접종일이나 사료 교체 시기 기록.
  - **싫어하는 것:** 병원 방문 시 유용한 정보가 되므로 추가하는 것을 적극 추천합니다.

#### 4. 기록하기 (포스팅 시스템)

- **기능:** 카메라/앨범 접근 권한을 얻어 사진 업로드 + 제목 + 상세 내용 작성.
- **메인화면 피드:** '최근 기록' 섹션에 썸네일, 날짜, 요약 글 노출.
- **상세 보기:** 썸네일 클릭 시 **모달(Modal)** 창 혹은 **디테일 페이지** 오픈.
- **💡 더 좋은 방법:**
  - 모달은 가벼운 확인에 좋지만, 글이 길어질 경우를 대비해 **별도 상세 페이지(Detail Page)**로 이동하고 '공유하기'나 '수정하기' 기능을 넣는 것이 확장성이 더 좋습니다.

#### 5. 추억보기 (타임라인)

- **기능:** 그동안 올린 모든 '기록하기' 글을 날짜별(역순)로 나열.
- **구성:** 인스타그램 피드 방식(그리드)이나 페이스북 방식(리스트) 중 선택 가능. 캘린더 뷰를 추가하여 특정 날짜의 기록을 쉽게 찾게 하면 더욱 좋습니다.

#### 6. 글 남기기 (방명록 & AI 코멘트)

- **기능:** 반려동물에게 전하는 편지나 그리움을 담은 글 작성.
- **수수료/유료 모델:** 유료 결제 시, 작성한 글과 반려동물의 정보(성격, 별명 등)를 **LLM(ChatGPT 등) API**에 전달하여, 마치 **반려동물이 하늘나라나 옆에서 대답해 주는 듯한 따뜻한 코멘트**를 생성.
- **효과:** 펫로스 증후군을 겪는 사용자나 교감을 원하는 사용자에게 강력한 정서적 가치 제공.

#### 7. 네비게이션 및 더보기 메뉴

- **하단 바 구성:** [홈] - [추억보기(타임라인)] - [방명록(글 남기기)] - [더보기].
- **더보기(슬라이드 메뉴):**
  - 고객센터, 공지사항.
  - **반려견 정보 수정:** 회원가입 때 입력한 정보를 언제든 수정 가능하게 구현 (몸무게는 자주 변하므로 필수).

#### 8. UI/UX 개선 제안: 기록하기 버튼 위치

- **제안하신 내용:** 기록하기 버튼을 최근 기록 맨 아래로 이동.
- **피드백:** 기록이 많아지면 사용자가 버튼을 찾기 위해 한참 내려야 하는 불편함이 생길 수 있습니다.
- **💡 더 좋은 아이디어:**
  - **플로팅 액션 버튼 (FAB):** 화면 우측 하단에 떠 있는 '+' 버튼을 두어 어떤 화면에서도 바로 기록할 수 있게 합니다.
  - **중앙 탭 배치:** 하단 네비게이션 바 정중앙에 '기록하기' 버튼을 크게 배치하는 것이 요즘 트렌드이며 접근성이 가장 좋습니다.

### ✨ 전체적인 구조 정리 (요약)

| 구분              | 주요 기능 및 특징                                                       |
| :---------------- | :---------------------------------------------------------------------- |
| **로그인/프로필** | 닉네임 맞춤형 인사, 반려견 사진 전역 연동                               |
| **반려견 등록**   | 이름, 생일, 몸무게, 태그(#좋아함), 싫어함, 취미 + 함께한 날짜 자동 계산 |
| **기록(포스팅)**  | 사진+제목+내용, 메인 피드 썸네일 노출, 클릭 시 상세 페이지              |
| **타임라인**      | 과거 기록 전체 보기 (앨범 형태)                                         |
| **커뮤니티/AI**   | 반려동물에게 쓰는 편지 + **(유료) AI 자동 코멘트 답장**                 |
| **설정/관리**     | 더보기 메뉴를 통한 정보 수정 및 고객 지원                               |
| **UX 최적화**     | 기록하기 버튼은 접근성이 좋은 고정 위치(중앙 탭 등) 추천                |
