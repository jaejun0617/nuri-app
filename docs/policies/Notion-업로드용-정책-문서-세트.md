# NURI Notion 업로드용 정책 문서 세트

최종 업데이트: 2026-04-02
용도: Notion 직접 반영이 불가능할 때 운영자가 그대로 옮길 수 있도록 정리한 업로드 가이드

## 1. 권장 페이지 구조

- 최상위 페이지: `NURI 정책 문서`
- 하위 페이지 1: `NURI 이용약관`
- 하위 페이지 2: `NURI 개인정보처리방침`
- 하위 페이지 3: `NURI 계정 삭제 / 탈퇴 안내`
- 하위 페이지 4: `NURI 커뮤니티 운영정책`

## 2. 각 페이지의 repo source of truth

- 이용약관
  - `docs/policies/이용약관.md`
- 개인정보처리방침
  - `docs/policies/개인정보처리방침.md`
- 계정 삭제 / 탈퇴 안내
  - `docs/policies/계정-삭제-탈퇴-안내.md`
- 커뮤니티 운영정책
  - `docs/policies/커뮤니티-운영정책.md`

## 3. Notion 페이지 상단 권장 메타

각 페이지 상단에 아래 3줄을 동일하게 두는 것을 권장합니다.

- 문서 상태: 작성 완료
- 외부 공개 상태: 검증 대기
- 최종 업데이트: 2026-04-02

## 4. 붙여넣기 순서

1. 각 source 파일의 제목부터 본문 끝까지 그대로 붙여넣습니다.
2. `운영자 확인 필요 사항`은 접은글(toggle)로 바꿔도 되지만, 삭제하지는 않습니다.
3. 공개 URL을 실제로 연결하기 전까지 `적용일`, `사업자 정보`, `문의처`를 임의 확정하지 않습니다.
4. 페이지 공유 설정을 바꾼 뒤에는 실제 공개 URL을 별도로 기록합니다.

## 5. 앱 링크 연결 준비 메모

- 회원가입에 연결할 최종 페이지
  - 이용약관
  - 개인정보처리방침
  - 마케팅 수신 안내는 별도 정책 공개 여부를 운영자가 최종 결정해야 함
- More / 계정 삭제에 연결할 최종 페이지
  - 계정 삭제 / 탈퇴 안내

## 6. 업로드 후 반드시 남길 기록

- Notion page title
- page URL
- 공개/비공개 여부
- 앱 링크 연결 대상 여부
- 실제 열림 검증 기기와 날짜
