# NURI 계정 삭제 / 탈퇴 안내

문서 상태: 작성 완료
외부 공개 상태: 공개 URL 운영 중, 재반영 필요
사용자 검증 상태: PO 승인 완료
최종 업데이트: 2026-04-03

## 1. 이 문서의 목적

이 문서는 NURI 앱에서 회원탈퇴 또는 계정 삭제를 요청할 때 실제로 어떤 처리 단계가 존재하는지, 무엇이 즉시 끝나고 무엇이 후속 정리로 이어질 수 있는지를 설명합니다.

## 2. 계정 삭제 요청의 의미

- 앱에서 `회원탈퇴`를 누른다고 해서 모든 데이터 정리가 즉시 끝나는 것은 아닙니다.
- NURI는 `삭제 요청 접수`와 `최종 정리 완료`를 구분해 처리합니다.
- 네트워크 상태나 backend 응답 상태에 따라 사용자는 중간 상태를 먼저 보게 될 수 있습니다.
- 운영자는 즉시 탈퇴 또는 일정 유예기간 후 자동 탈퇴와 같은 계정 종료 방식을 운영 정책으로 둘 수 있으며, 구체 절차와 적용 방식은 서비스 화면 또는 별도 안내에 따릅니다.

## 3. 현재 앱이 구분하는 주요 상태

현재 코드 기준으로 계정 삭제는 다음 상태를 가질 수 있습니다.

- `requested`
  - 삭제 요청이 접수됨
- `in_progress`
  - 삭제 처리 진행 중
- `db_deleted`
  - 주요 DB 삭제가 먼저 진행된 상태
- `cleanup_pending`
  - 스토리지 또는 후속 정리가 남은 상태
- `completed`
  - 처리 완료
- `completed_with_cleanup_pending`
  - 주요 삭제는 끝났지만 일부 cleanup 확인이 남은 상태
- `unknown_pending_confirmation`
  - timeout 또는 응답 지연으로 최종 상태를 아직 확정하지 못한 상태

## 4. 삭제 대상이 되는 정보

계정 삭제 시 다음 정보는 삭제 대상이 됩니다.

- 계정과 연결된 프로필 정보
- 닉네임과 사용자 개인화 정보
- 반려동물 프로필
- 기록과 기록 이미지
- 일정 정보
- 커뮤니티 개인 게시글, 댓글, 개인화 데이터
- 장소/여행 저장 및 개인 제보 데이터

## 5. 즉시 끝나지 않을 수 있는 항목

아래 항목은 비동기 후처리로 이어질 수 있습니다.

- 스토리지 파일 삭제
- cleanup queue 처리
- 운영 확인이 필요한 예외 상태 정리

따라서 `cleanup pending` 또는 유사한 상태는 성공 완료와 동일한 의미가 아닙니다.

## 6. 남을 수 있는 운영 증적

다음 범위의 데이터는 운영 목적상 그대로 두거나, 식별자를 제거한 뒤 남길 수 있습니다.

- 이용약관 / 개인정보처리방침 / 마케팅 수신 동의 이력
- 신고 접수 및 moderation 관련 운영 증적
- 시스템 오류 및 abuse 대응 추적 정보

이 경우에도 직접 식별자는 제거하거나, 더 이상 일반 사용자 계정과 직접 연결되지 않도록 처리하는 것이 원칙입니다.

현재 운영 정책 방향상 신고 내역, 커뮤니티 제재(숨김) 기록 등 일부 부정이용 방지 목적 정보는 비식별화 후 1년간 보관할 수 있으며, 서비스 방문(로그인) 기록은 관련 법령에 따라 3개월간 보관할 수 있습니다. 자세한 기준은 `개인정보처리방침`을 따릅니다.

## 7. unknown 상태 안내

- 앱이 삭제 요청 후 즉시 최종 응답을 받지 못하면 `unknown pending confirmation`처럼 운영 확인이 필요한 상태가 될 수 있습니다.
- 이 상태는 삭제가 실패했다는 의미와 같지 않으며, 반대로 삭제가 완전히 끝났다는 의미도 아닙니다.
- 운영자는 backend 상태를 별도로 확인해야 할 수 있습니다.

## 8. 계정 삭제 후 로그인과 복구

- 계정 삭제가 완료되면 기존 계정으로 다시 로그인할 수 없을 수 있습니다.
- 계정 삭제 후 동일 이메일의 재사용은 허용될 수 있으며, 구체적인 재가입 가능 시점은 운영 정책 및 처리 완료 상태에 따라 달라질 수 있습니다.
- 현재 앱과 backend는 `즉시 탈퇴`와 `일정 유예기간 후 자동 탈퇴`를 동시에 구현한 상태가 아니므로, 실제 적용 절차는 서비스 화면 또는 별도 안내를 따릅니다.

- 법정 보관 의무에 따른 예외 항목과 기간의 최종 문안
