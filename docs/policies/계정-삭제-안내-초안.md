# NURI 계정 삭제 안내 초안

상태: 초안

용도: 앱 연결용 초안

주의: 이 문서는 운영/법무 확정 전 구조 문서다. 최종 정책 본문으로 간주하지 않는다.

## 1. 삭제 요청의 의미

- 계정 삭제는 버튼을 누르는 즉시 모든 정리가 끝난다는 의미가 아니다.
- 앱은 삭제 요청 접수와 최종 정리 완료를 구분해서 다룬다.

## 2. 즉시 완료와 비동기 정리 가능성

- 계정과 개인 데이터의 주요 삭제는 먼저 처리될 수 있다.
- 스토리지 파일 정리와 일부 후처리는 비동기로 이어질 수 있다.

## 3. 개인 콘텐츠 삭제 원칙

- 프로필, 반려동물 정보, 기록, 일정, 커뮤니티 개인 콘텐츠, 개인화 데이터는 삭제 대상이 될 수 있다.

## 4. 익명화 및 비식별 보관 원칙

- 신고/운영 증적 일부는 사용자 식별자를 제거한 뒤 보관될 수 있다.
- 동의 이력은 직접 식별 연결을 끊고 비식별 보관될 수 있다.

## 5. storage cleanup 가능성

- 업로드된 파일 정리는 별도 cleanup 흐름으로 이어질 수 있다.
- cleanup pending 상태에서는 일부 파일 정리가 남아 있을 수 있다.

## 6. unknown 상태와 운영 확인

- 네트워크 지연이나 timeout이 발생하면 요청 완료 여부가 즉시 확정되지 않을 수 있다.
- 이 경우 운영 확인이 필요한 상태로 남을 수 있다.

## 7. 문의처

- 문의 채널: 미확정
- 운영/법무 확정 후 외부 공개 문서에 반영한다.
