# NURI 정책 문서 운영자 입력값 체크리스트

최종 업데이트: 2026-04-03
문서 목적: 정책 문서 최종 운영/법무 입력값과 Notion 재반영 전제조건을 잠그기 위한 입력 정리 문서
현재 상태: repo 최신 / Notion 재반영 대기 / 법무 확정본 미완료
다음 기술 태스크 상태: 계속 중단

## 1. 현재 잠긴 상태

- `이용약관`, `개인정보처리방침`, `계정 삭제 / 탈퇴 안내`는 `작성 완료 + Notion 1차 반영 완료 + 앱 링크 연결 완료 + Android 실기기 검증 완료 + PO 승인 완료` 상태다.
- 이번 턴 기준 repo 정책 문서는 운영자 2차/3차 입력값과 최종 운영/법무 입력 3개를 반영한 최신본이다.
- 현재 Notion 공개본은 이번 repo 변경을 아직 반영하지 않아 stale 상태다.
- 이번 턴은 앱 코드, DB, 테스트, QA, 기술 태스크를 건드리지 않는다.

## 2. 최우선 입력 3개 잠금

### 2-1. 사업자명 또는 법적 운영 주체 병기

- 현재 상태
  - 공개 문서 운영 주체 표기를 `펫 누리(Pet Nuri) 운영자 신재준`으로 확정했다.
- 미확정 이유
  - 사업자 등록은 아직 진행 전이므로 사업자명, 사업자등록번호 병기 형식은 미완료다.
- 성격
  - 운영 + 법무
- 내가 답해야 했던 질문
  - 외부 공개 문서에 개인 실명 또는 사업자명을 지금 병기할 것인지
  - 현재는 서비스형 운영 주체 표기만 유지할 것인지
- 답변 예시 형식
  - `운영 주체 표기: 펫 누리(Pet Nuri) 운영자 신재준`
  - `사업자 등록 전까지 개인 실명 병기 유지`
- 이번 턴 확정 답변
  - `펫 누리(Pet Nuri) 운영자 신재준`
- 반영 문서
  - `이용약관.md`
  - `개인정보처리방침.md`
  - `계정-삭제-탈퇴-안내.md`
  - `커뮤니티-운영정책.md`
  - `마케팅-정보-수신-동의서.md`
- Notion 재반영 필요 여부
  - 예

### 2-2. 마케팅 정보 수신 동의서 공개 여부

- 현재 상태
  - 독립 문서 `마케팅-정보-수신-동의서.md`는 작성 완료 상태이며, 공개 대상 문서로 전환했다.
- 미확정 이유
  - 실제 Notion 공개와 앱 링크 연결은 아직 실행하지 않았고, 앱 draft 경로는 여전히 기존 mirror 파일을 가리킨다.
- 성격
  - 운영 + 법무 + 기술 후속
- 내가 답해야 했던 질문
  - 지금 공개 문서 세트에 포함할지
  - 실제 발송 채널과 철회 방식이 잠긴 뒤 공개할지
- 답변 예시 형식
  - `마케팅 정보 수신 동의서: 즉시 공개 대상`
  - `앱 링크 연결 대상 포함, 실제 코드 연결은 후속 허용 턴`
- 이번 턴 확정 답변
  - `즉시 공개 대상으로 전환`
- 반영 문서
  - `마케팅-정보-수신-동의서.md`
  - `마케팅-수신-안내-초안.md`
  - `Notion-반영-검증-기록.md`
  - 정책 링크 준비/작업 기록 문서
- Notion 재반영 필요 여부
  - 예

### 2-3. Supabase / Sentry 국외 이전 및 처리위탁 고지

- 현재 상태
  - 개인정보처리방침에 `처리위탁`과 `국외 이전`을 모두 명시하는 방향으로 확정 문구를 반영했다.
- 미확정 이유
  - 법정 고지 형식의 최종 세부 조정은 아직 법무 검토가 남아 있다.
- 성격
  - 운영 + 법무
- 내가 답해야 했던 질문
  - `처리위탁`으로만 둘지, `국외 이전`으로만 둘지, 둘 다 병기할지
  - 국가/업무/보관기간을 어느 수준까지 공개할지
- 답변 예시 형식
  - `Supabase, Inc.: 처리위탁 + 국외 이전 모두 명시`
  - `Sentry: 처리위탁 + 국외 이전 모두 명시`
- 이번 턴 확정 답변
  - `처리위탁과 국외 이전 둘 다 명시`
  - `Supabase, Inc. / 미국 / 클라우드 서버 인프라 제공 및 데이터 보관 / 회원 탈퇴 시까지`
  - `Sentry / 미국 / 앱 오류 모니터링 및 안정성 분석 / 회원 탈퇴 시까지`
- 반영 문서
  - `개인정보처리방침.md`
  - `정책-문서-정합성-점검.md`
  - `Notion-반영-검증-기록.md`
- Notion 재반영 필요 여부
  - 예

## 3. Notion 재반영 대기 범위

- 재반영 대상
  - `NURI 이용약관`
  - `NURI 개인정보처리방침`
  - `NURI 계정 삭제 / 탈퇴 안내`
  - `NURI 커뮤니티 운영정책`
  - `NURI 마케팅 정보 수신 동의서`
- 재반영 대기 이유
  - repo 문서는 최신이지만, 현재 Notion 공개본은 운영 주체 실명 병기, 마케팅 공개 대상 전환, 처리위탁/국외 이전 확정 방향을 아직 반영하지 않았다.
- 지금 잠근 상태
  - repo 문서: 최신
  - Notion 공개본: 재반영 대기
  - 법무 확정본: 미완료
  - 다음 기술 태스크: 계속 중단

## 4. 남은 법무 확인 항목

- 사업자 등록 이후 표기 형식과 병기 범위
- 개인정보처리방침의 처리위탁/국외 이전 표 최종 법정 형식
- 미성년자 문안의 최종 표현
- 보관기간 문구형과 표 형식의 최종 정리
- 커뮤니티 제재/이의제기 문안의 최종 적정성

## 5. 기술 후속 메모

- `documents.ts`는 이번 턴 수정 금지 범위라 건드리지 않았다.
- 따라서 마케팅 문서는 `즉시 공개 대상`으로 정책 문서 기준이 잠겼지만, 실제 앱 링크 연결은 다음 허용 턴의 별도 작업이다.
- 커뮤니티 운영정책 인앱 노출은 이미 후속 Task로 등록돼 있으며, 이번 턴에도 구현하지 않는다.
